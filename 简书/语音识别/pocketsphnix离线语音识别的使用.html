pocketsphnix离线语音识别的使用

1 .安装SpeechRecognition模块

pip install SpeechRecognition

2 ,使用recognize_sphinx()离线语音识别，安装PocketSphinx

pip  install pocketsphinx

注意:如果安装不成功的话需要错误提示安装whl文件

切换到whl文件所在的文件夹下用pip进行安装即可.

pip   install   pocketsphinx-0.1.15-cp37-cp37m-win_amd64.whl

3.测试代码

# -*- coding: utf-8 -*-

"""

Created on Sun Feb 24 22:30:23 2019

@author: zangz

"""

import speech_recognition as sr

# obtain audio from the microphone

r = sr.Recognizer() 

# print(r)

for i in range(122,200):

    str1='S0002/BAC009S0002W0'+str(i)+'.wav'

    harvard = sr.AudioFile(str1)

#    print(harvard)

    with harvard as source:

        audio = r.record(source)

    # recognize speech using Sphinx

    try:

        print("Sphinx thinks you said " + r.recognize_sphinx(audio,language="zh_CN"))

    except sr.UnknownValueError:

        print("Sphinx could not understand audio")

    except sr.RequestError as e:

        print("Sphinx error; {0}".format(e))

注意直接识别 中文是不支持的需要下载中文的模型.
